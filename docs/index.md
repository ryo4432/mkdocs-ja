# HOME

## 概要

mkdocsの使い方を自分用のメモとして残します。  
mkdocsはマークダウンをhtmlの静的サイトにするためのツールです。既存のマークダウンをそのまま使えるのが利点です。  
マークダウンとサイト構成を定義するyamlファイルを定義するだけでお手軽にいい感じのサイトを作れます。
インストールについてはDockerが使えれば大丈夫です。
